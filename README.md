# jules-test-drive

## Pink Calculator Web App

A web-based calculator application inspired by a pink physical calculator. This project implements a fully functional calculator with a custom design and dark mode support.

### Features

*   **Arithmetic Operations:** Addition, Subtraction, Multiplication, Division.
*   **Special Functions:**
    *   `AC`: All Clear
    *   `CE`: Clear Entry
    *   `→`: Backspace
    *   `√`: Square Root
    *   `%`: Percentage
*   **Memory Functions:**
    *   `M+`: Add to memory
    *   `M-`: Subtract from memory
    *   `MRC`: Memory Recall / Memory Clear (Click once to recall, twice to clear)
*   **Visual Design:**
    *   Pink aesthetic mimicking a physical calculator.
    *   3D button effects with shadows.
    *   LCD-style display.
*   **Dark Mode:**
    *   Toggle switch to enable Dark Mode.
    *   Persists user preference using Local Storage.

### Usage

1.  Clone the repository.
2.  Open `index.html` in your web browser.
3.  Use the mouse to click buttons.

### Files

*   `index.html`: The HTML structure of the calculator.
*   `style.css`: The styling, including CSS variables for theming.
*   `script.js`: The logic for calculator operations and theme toggling.
*   `calculator.py`: A simple Python calculator script (separate from the web app).
